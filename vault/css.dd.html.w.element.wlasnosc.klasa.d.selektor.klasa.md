# CSS

## Dołączenie CSS do elementu HTML za pomocą klasy HTML

```html
<div class="main-content">
    <h1>
        Welcome to {{ title }}!
    </h1>
</div>
```

* * *

```css
.main-content {
    padding: 20px;
    font-family: sans-serif;
}
```
